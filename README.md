# vuejs-spa-dev-env
Vue.jsでSPA(Single Page Application)作るための開発環境的なやつ

とりあえずbuildして起動するまで書いておきます。詳細は徐々に。

## Build

### step 1

    npm install

or

    yarn install

### step 2

    gulp build


### step 3

    gulp

Google Chromeが立ち上がってサイトが表示されます。
この状態でsrc内のファイルに変更があると自動でブラウザが更新されます。
